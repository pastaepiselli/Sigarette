# Sigarette

Microservizio che ritorna tutte le sigarette italiane con:
### Brand:
Stringa con il nome del brand.

### Prezzo
Prezzo indicato con un real in db, in java double.

### Nicotina 
Nicotina indicata con un real in db, in java double.

### Catrame
Catrame indicato con un real in db, in java double.

### Tipo
Per il tipo pensavo di indicare il colore ma sto ancora capendo cosa indicare :P.

### Descrizione
Breve descrizione del sapore della sigaretta.

### URL_IMMAGINE
Da capire dove trovare le immagine, in questo campo viene inserito un link ad un servizio di cloud gratuito, [Cloudinary](https://cloudinary.com/)
in cui sono salvate le foto.
